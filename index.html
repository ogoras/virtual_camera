Help

Arrow keys - translate forward, backward, left, right | PageUp/PageDown - translate up/down | A/D - Rotate left/right (Yaw -/+) | W/S - Rotate up/down (Pitch +/-) | Q/E - Rotate counterclockwise/clockwise (Roll -/+) | +/- - zoom in/out | 0 - reset zoom | Home - reset view | Escape - exit | H - toggle help | Z - toggle zoom display
